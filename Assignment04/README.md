# Flexible Notification System

## How to Compile
download all files from src directory
# From Command Prompt:
cd src (or what ever directory you put the files in) 
javac *.java

## How to Run
java Main

# Successful Run will say the following after Main is executed:
Email sent: This is your email.
SMS message sent: This is your SMS message.

Message Log:
- This is your email.
- This is your SMS message.
